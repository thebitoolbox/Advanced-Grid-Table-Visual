# Privacy Policy for Advanced Grid Table

**Last updated: March 22, 2026**

This privacy policy applies to the "Advanced Grid Table" custom visual for Microsoft Power BI.

### 1. Data Collection and Usage
The Advanced Grid Table visual operates entirely within the user's client environment (web browser or Power BI Desktop application). **It does not collect, store, transmit, or share any personal, organizational, or telemetry data with any external servers or third parties.**

### 2. Data Processing
All data processing, including conditional formatting, column resizing, sorting, and user-input notes, happens locally on the user's machine. The visual only reads the data explicitly provided to it through the Power BI data view mappings to render the table.

### 3. Operational Notes Feature
Any text entered by users into the "Notes" column is stored temporarily in the visual's local state and/or the local browser session. It is not transmitted to the developer, Microsoft, or any external database.

### 4. Contact
If you have any questions or concerns about this privacy policy, please open an issue on our GitHub repository.
